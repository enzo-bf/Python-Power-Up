# ⚡ Python Power Up

Projeto desenvolvido em Python com foco em **automação de tarefas**, realizando processos repetitivos de forma rápida e eficiente.

O objetivo é praticar a utilização de bibliotecas para automatizar atividades, aumentando a produtividade e reduzindo o trabalho manual.

---

## 🚀 Funcionalidades

- Automação de tarefas
- Controle do teclado e mouse
- Abertura automática de programas
- Preenchimento de informações
- Execução de processos repetitivos

---

## 🛠️ Tecnologias Utilizadas

- Python 3
- PyAutoGUI
- Jupyter Notebook

---

## 📂 Estrutura do Projeto

```
Python-Power-Up/
│
├── automacao.py
├── auxiliar.ipynb
└── README.md
```

---

## ▶️ Como Executar

1. Clone este repositório:

```bash
git clone https://github.com/enzo-bf/Python-Power-Up.git
```

2. Instale as dependências:

```bash
pip install pyautogui
```

3. Execute o arquivo principal:

```bash
python automacao.py
```

---

## 🎯 Objetivo

Este projeto foi desenvolvido para praticar conceitos de automação utilizando Python, permitindo automatizar tarefas repetitivas por meio do controle do teclado e do mouse.
